# Profile Anomaly Detection & Risk Scoring System

A web-based dashboard for analyzing social media profiles and identifying potentially suspicious accounts using engagement patterns and rule-based risk scoring.

## Overview

This project evaluates social media profiles based on observable attributes such as follower count, following count, and account activity. It applies structured logic to detect irregular patterns commonly associated with low-quality or potentially fake profiles.

## Features

- Follower-to-following ratio analysis  
- Engagement-based risk scoring  
- Detection of irregular profile patterns  
- Clean and responsive dashboard interface  
- Instant classification (Real / Suspicious)

## How It Works

The user provides basic profile metrics through the interface.  
The system evaluates these inputs using predefined conditions and assigns a risk label based on detected anomalies in engagement patterns.

## Limitations

- Uses rule-based logic (no machine learning model)  
- Requires manual input data  
- Not integrated with live social media APIs
